The python program provided approximates smallest eigenvalue to certain Sturm-Liouville problems with Dirichlet-Neumann boundary conditions. 
The code is an implementation of a Prüfer transform based shooting method detailed in the paper. For the algorithm implemented to work, it is
required that both p and q are second order continuously differentiable and strictly positive. Here, a few example problems are provided in the 
code and paper but the code can easily be modified to handle other problems. To use the code, python, numpy and sympy must be installed.
